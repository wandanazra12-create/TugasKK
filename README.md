# Tugas KK - Sistem Manajemen Kata dan Kalimat (Laravel)

Proyek ini adalah aplikasi berbasis web yang dibangun menggunakan framework **Laravel 11** untuk memenuhi tugas mata pelajaran **KK3 (Kompetensi Keahlian 3)**. Aplikasi ini fokus pada manajemen data huruf, kata, dan kalimat.

## 🚀 Fitur Utama
- **Manajemen Huruf:** Tambah, edit, dan hapus data huruf.
- **Manajemen Kata:** Pengolahan data kata beserta detailnya.
- **Manajemen Kalimat:** CRUD (Create, Read, Update, Delete) untuk data kalimat.
- **UI/UX:** Menggunakan layout Blade yang konsisten dan responsif dengan tema yang dikustomisasi.

## 🛠️ Teknologi yang Digunakan
- **Framework:** [Laravel 11](https://laravel.com)
- **Database:** MySQL / SQLite
- **Frontend:** Blade Templating, CSS, JS (Vite), FontAwesome

## 📋 Prasyarat Instalasi
Sebelum menjalankan aplikasi, pastikan Anda sudah menginstal:
